# Prompty do NanoBanana - Materialy Marketingowe Asystent IO

## Kontekst dla wszystkich promptow

**Aplikacja:** Asystent IO (IR Assistant)
**Cel:** Pomoc osobom z insulinoopornoscia w codziennych wyborach zywieniowych
**Kolorystyka:** Zielony (#2E7D52), bialy, jasna zielen
**Styl:** Nowoczesny, czysty, przyjazny, medyczny ale nie kliniczny
**Logo:** Lisc z motywem elektroniki/tech (polaczenie natury i technologii)
**Ton:** Wspierajacy, edukacyjny, bez oceniania

---

## PROMPT 1: Hero Banner dla Landing Page (Desktop)

```
Create a modern, clean hero banner for a health app landing page.

Subject: A smartphone showing a food scanning app interface with a green circular score indicator (showing "95/100"). The phone should be slightly angled, floating with a subtle shadow.

Background: Soft gradient from white to very light mint green (#E8F5E9). Abstract leaf shapes subtly visible in the corners.

Style: Minimalist, SaaS landing page aesthetic, professional but friendly. No text on the image.

Colors: Primary green #2E7D52, white, soft mint accents.

Dimensions: 1920x800px, landscape orientation.

Additional elements: Small floating food icons (apple, bread, barcode) around the phone, very subtle and semi-transparent.
```

## PROMPT 3: Feature Icon Set (6 icons)

```
Create a set of 6 matching feature icons for a health/food app landing page.

Icons needed:
1. Barcode scanning - barcode with scan lines
2. OCR/Camera - camera lens with text recognition
3. Menu analysis - restaurant menu with checkmarks
4. Food/Dish analysis - plate with food items
5. Quiz/Learning - brain with lightbulb
6. Privacy/Shield - shield with lock

Style: Line icons with filled accents, consistent 3px stroke weight, rounded corners.

Colors: Primary green #2E7D52, with white or light mint backgrounds in circles.

Each icon: 128x128px, transparent background.

Style reference: Modern SaaS feature icons, Phosphor Icons aesthetic.
```

4. **Rozdzielczosc** - Generuj w najwyzszej mozliwej jakosci
